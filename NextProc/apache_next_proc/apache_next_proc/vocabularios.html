VocabulariosVocabularies

A continuación se pueden consultar los vocabularios utilizados dentro de la iniciativa nextProcurement. Por el momento, usamos únicamente la ontología OCDS, pero en el futuro podría integrarse también la ontología eProcurement. The vocabularies used within the nextProcurement initiative can be found below. For now, we only use the OCDS ontology, but in the future, the eProcurement ontology could be integrated as well.

NombreName DescripciónDescription TypeTipo TemaTopic URL DocumentaciónDocumentation
OCDS La ontología OCDS permite representar todo el proceso de contratación pública, incluyendo concenptos como ``Tender'', ``Bid'' o ``Award''. Esta ontología se desarrolló durante el proyecto europeo TheyBuyForYou se basa en el modelo de datos delThe OCDS ontology allows to represent the whole public procurement process, including concepts such as ``Tender'', ``Bid'' or ``Award''. This ontology was developed during the European project TheyBuyForYou and is based on the data model in the Open Contracting Data Standard. OntologyOntología Public ProcurementAdministración Pública https://github.com/TBFY/ocds-ontology https://mnavasloro.github.io/documentationOCDS/
eProcurement The eProcurement ontology makes available in an open, structured and machine-readable format data about public procurement, covering it from end to end, i.e. from notification, through tendering to awarding, ordering, invoicing and payment. The eProcurement ontology makes available in an open, structured and machine-readable format data about public procurement, covering it from end to end, i.e. from notification, through tendering to awarding, ordering, invoicing and payment. OntologyOntología Public ProcurementAdministración Pública https://github.com/OP-TED/ePO https://eprocurementontology.github.io/index.html
