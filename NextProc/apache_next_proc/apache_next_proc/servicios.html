Services createdServicios desarrollados

A continuación se listan las distintas herramientas desarrolladas para procesar la información relativa a las licitaciones el contexto del proyecto. The different tools developed to process tender-related information in the context of the project are listed below.

Clasificador de códigos CPVCPV code classifier

El clasificador de códigos CPV recibe la descripción de una oferta de licitación y devuelve las divisiones más probables con las que anotarla. El modelo que obtiene mejores resultados por el momento es un fine-tuning del modelo RoBERTa entrenado con el corpus de la BNE, y está disponible en HuggingFace, y se irá actualizando a medida que se refine la tarea. Asimismo, el listado con los notebooks y los datasets utilizados se puede encontrar en el repositorio de GitHub. En la siguiente demo puede probarse el servicio.
 The CPV code classifier receives the description of a tender and returns the most likely divisions with which to annotate it. The best performing model at the moment is a fine-tuning of the RoBERTa model trained on the BNE corpus, and is available at HuggingFace. It will be further improved. Additionally, the notebooks and dataset used can be found in the GitHub repository. The following demo shows the service.

Explorador de licitacionesTender Explorer

Esta herramienta permite explorar distintas licitaciones. Disponible en http://tbfy.librairy.linkeddata.es/explorer/ This tool allows to explore different tenders.It is available at http://tbfy.librairy.linkeddata.es/explorer/


Búsqueda de licitacionesTender search

Esta herramienta permite buscar licitaciones similares presentes en el grafo de conocimiento a la descripción introducida por el usuario. El servicio puede probarse en la siguiente dirección https://tbfy.librairy.linkeddata.es/search-api/api.html This tool allows to search in the knowledge graph tenders similar to the description provided by the user. The service can be tested in the following link https://tbfy.librairy.linkeddata.es/search-api/api.html

Servicio de clusteringClustering service

The LibrAIry service classifies the tenders in different clusters trained on the JCR-Acquis corpus. El servicio LibrAIry clasifica las licitaciones en distintos clusters entrenados sobre el corpus JCR-Acquis.
