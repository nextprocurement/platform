Procesamiento de los datos

Para alimentar el grafo de conocimiento se ha diseñado una pipeline de procesamiento de datos que, tomando los datos de origen de las plataformas de datos abiertos PLACE y TED, es capaz de generar una base de datos RDF sobre la que poder realizar consultas.

Para la realización de este procesamiento de datos se ha tomado como partida el proyecto They Buy For You, reutilizando el procesamiento de datos orientándolo a las necesidades de nextProcurement.

El procesamiento de datos de They Buy For You fue modificado incorporando el motor para la construcción de grafos de conocimiento Morph-KGC y el servidor para el almacenamiento de los datos enlazados Virtuoso Open Source, así como empleando la herramienta para la generacion de archivos de mapeado YARRRML.

El resultado es una pipeline con una API que permite su adaptacion a nuevas entradas de datos de fuentes diversas. El resultado puede observarse en el servidor de Virtuoso y en su Sparql Endpoint.
