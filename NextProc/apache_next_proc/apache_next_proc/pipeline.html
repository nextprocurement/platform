Data PipelineProcesamiento de datos

To feed the knowledge graph, a data processing pipeline has been designed. This pipeline takes the source data from the PLACE and TED open data platforms and is able to generate an RDF database on which queries can be made. Para alimentar el grafo de conocimiento se ha diseñado una pipeline de procesamiento de datos que, tomando los datos de origen de las plataformas de datos abiertos PLACE y TED, es capaz de generar una base de datos RDF sobre la que poder realizar consultas.

Para la realización de este procesamiento de datos se ha tomado como partida el proyecto They Buy For You, reutilizando el procesamiento de datos orientándolo a las necesidades de nextProcurement.

They Buy For You data pipeline was modified by incorporating the knowledge graph construction engine Morph-KGC and the linked data storage server Virtuoso Open Source, as well as using the tool for generating the YARRRML mappings. El procesamiento de datos de They Buy For You fue modificado incorporando el motor para la construcción de grafos de conocimiento Morph-KGC y el servidor para el almacenamiento de los datos enlazados Virtuoso Open Source, así como empleando la herramienta para la generacion de archivos de mapeado YARRRML.

The result is a pipeline with an API that allows to be adapted to new diverse data sources. The result can be explored in the Virtuoso server and in its Sparql Endpoint. El resultado es una pipeline con una API que permite su adaptacion a nuevas entradas de datos de fuentes diversas. El resultado puede observarse en el servidor de Virtuoso y en su Sparql Endpoint.
